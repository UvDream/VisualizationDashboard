# 🎨 可视化大屏编辑器

一个功能强大的可视化大屏编辑器，支持拖拽式组件布局、实时预览、丰富的图表组件和炫酷的 3D 效果。适用于数据大屏、监控面板、展示页面等场景。

## 🛠️ 技术栈

- **前端框架**: React 19 + TypeScript
- **构建工具**: Vite (Rolldown)
- **UI 组件**: Ant Design 6
- **图表库**: ECharts 6
- **拖拽功能**: React DnD
- **3D 渲染**: React Three Fiber + Three.js
- **状态管理**: React Context

## 🚀 功能特性

### 📊 丰富的组件库
- **图表组件** (15种): 折线图、柱状图、饼图、仪表盘、雷达图、散点图、地图、词云图、漏斗图、日历热力图、树形分布图等
- **UI 组件** (12种): 文本、按钮、输入框、选择器、开关、进度条、标签、徽标、头像、卡片、表格、滚动排名、轮播列表
- **3D 组件** (10种): 3D地球、粒子背景、3D魔方、DNA螺旋、3D波浪、3D环形、星系、时空隧道、矩阵雨、等离子球

- **布局组件**: 两栏布局、三栏布局、头部布局、侧栏布局

- **图标库**: 支持所有 Ant Design 图标

- **组合功能**: 支持组件组合和取消组合

- **复制粘贴**: 快速复制组件

### 📐 画布功能
- **自由缩放**: 支持画布缩放控制
- **标尺辅助**: 精确的标尺和网格
- **画布配置**: 自定义画布尺寸、背景色、背景图片
- **预览模式**: 一键切换预览模式

### 🔗 数据源支持
- **静态数据**: 支持手动配置数据
- **API 接口**: 支持 RESTful API 数据源
- **实时刷新**: 支持数据自动刷新
- **数据映射**: 灵活的数据字段映射

### 🎨 样式配置
- **属性面板**: 丰富的样式配置选项
- **实时预览**: 所见即所得的编辑体验
- **响应式**: 支持不同屏幕尺寸适配

## 🎮 使用指南

### 基本操作
1. **添加组件**: 从左侧组件面板拖拽组件到画布
2. **选择组件**: 点击画布上的组件进行选择
3. **移动组件**: 拖拽选中的组件进行移动
4. **调整大小**: 拖拽组件四角的控制点调整大小
5. **配置属性**: 在右侧属性面板配置组件样式和数据

### 高级功能
- **多选**: 按住 Ctrl/Cmd 点击多个组件，或框选多个组件
- **组合**: 选中多个组件后右键选择"组合"
- **图层操作**: 在图层面板中管理组件层级关系
- **数据绑定**: 在属性面板中配置数据源和 API 接口

### 快捷键
- `Ctrl/Cmd + Z`: 撤销
- `Ctrl/Cmd + Y`: 重做
- `Delete/Backspace`: 删除选中组件
- `Ctrl/Cmd + 点击`: 多选组件

## 🔧 自定义开发

### 添加新组件
1. 在 `types/index.ts` 中添加组件类型
2. 在 `config/defaultConfigs.ts` 中配置默认属性
3. 在 `ComponentPanel/index.tsx` 中注册组件
4. 在 `CanvasItem.tsx` 中实现渲染逻辑

### 扩展数据源
在 `utils/dataSource.ts` 中添加新的数据源类型和处理逻辑。
