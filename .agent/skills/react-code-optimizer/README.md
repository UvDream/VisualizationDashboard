# React Code Optimizer Skill

A comprehensive code analysis and optimization skill for React/TypeScript projects.

## Quick Start

```
Use react-code-optimizer to analyze my project
```

## What's Included

- **SKILL.md** - Complete skill documentation with usage instructions
- **examples/** - Example usage scenarios and sample outputs
- **templates/** - Templates for generating optimization reports
- **checklists/** - Step-by-step analysis checklists

## Features

✅ **Performance Analysis**
- Detects missing React.memo
- Identifies expensive re-renders
- Analyzes Context provider optimization
- Checks hook optimization (useCallback, useMemo)

✅ **Code Organization**
- Flags large files (>500 lines)
- Identifies code duplication
- Suggests component splitting
- Recommends custom hook extraction

✅ **Architecture Review**
- State management patterns
- Data flow analysis
- Error handling verification
- Side effect management

✅ **Type Safety**
- TypeScript usage review
- Identifies type issues
- Suggests type improvements

## Output

The skill generates:
1. **task.md** - Checklist of issues found
2. **implementation_plan.md** - Detailed optimization plan with code examples
