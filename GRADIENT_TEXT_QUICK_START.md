# 渐变文字组件 - 快速开始

## 🎯 30秒快速上手

### 1️⃣ 添加组件
- 打开可视化编辑器
- 在左侧组件库中找到"组件库"分类
- 找到"渐变文字"组件
- 拖拽到画布上

### 2️⃣ 编辑文本
- 选中组件
- 在右侧属性面板的"数据"选项卡中
- 在"文本内容"中输入你的文字

### 3️⃣ 调整样式
- 在右侧属性面板的"基础"选项卡中
- 展开"渐变配置"部分
- 调整以下属性：
  - **字体大小**：改变文字大小
  - **渐变类型**：选择线性或径向
  - **渐变颜色**：修改颜色数组
  - **文字阴影**：启用阴影效果

## 🎨 常用配置

### 彩虹渐变
```json
["#ff0000", "#ffff00", "#00ff00", "#00ffff", "#0000ff", "#ff00ff"]
```

### 火焰渐变
```json
["#ff0000", "#ff6600", "#ffff00"]
```

### 冰蓝渐变
```json
["#00ffff", "#0099ff", "#0033ff"]
```

### 紫色渐变
```json
["#ff00ff", "#9900ff", "#0033ff"]
```

## ⚙️ 常用参数

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 字体大小 | 32-48 | 标题用 48，正文用 24-32 |
| 字体粗细 | bold | 加粗效果更好 |
| 渐变类型 | linear | 线性渐变最常用 |
| 渐变角度 | 45 | 45° 效果最佳 |
| 文字阴影 | 启用 | 增强立体感 |
| 阴影模糊度 | 10-15 | 柔和效果 |

## 💡 使用技巧

### 技巧1：多行文字
在"文本内容"中使用换行符创建多行文字：
```
第一行文字
第二行文字
第三行文字
```

### 技巧2：调整行高
- 单行文字：行高 1.0-1.2
- 多行文字：行高 1.5-2.0

### 技巧3：字间距
- 紧凑：0-2px
- 正常：2-4px
- 宽松：4-8px

### 技巧4：对齐方式
- 标题：居中
- 左对齐：用于列表
- 右对齐：用于特殊排版

## ❓ 常见问题

### Q: 如何改变渐变方向？
A: 在"渐变角度"中调整数值（0-360°）

### Q: 如何添加更多颜色？
A: 在"渐变颜色"中修改数组，添加更多颜色值

### Q: 如何禁用阴影？
A: 关闭"文字阴影"开关

### Q: 支持哪些颜色格式？
A: 支持 HEX (#ff0000)、RGB (rgb(255,0,0))、RGBA (rgba(255,0,0,0.5))

### Q: 文字内容支持多少行？
A: 无限制，但建议不超过 5 行
