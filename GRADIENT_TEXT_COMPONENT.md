# 渐变文字组件使用指南

## 组件概述

渐变文字组件是一个特效文字组件，支持线性和径向渐变效果，可以创建炫彩文字效果。

## 功能特性

- **渐变类型**：支持线性渐变和径向渐变
- **渐变颜色**：支持多色渐变，可自定义颜色数组
- **文字阴影**：可选的文字阴影效果
- **文字样式**：支持字体大小、粗细、间距、行高等调整
- **文本对齐**：支持左对齐、居中、右对齐

## 使用示例

### 1. 基础渐变文字

在组件库中找到"渐变文字"组件，拖拽到画布上。默认配置已包含红绿蓝三色线性渐变。

### 2. 自定义渐变颜色

在属性面板的"渐变配置"中修改"渐变颜色"属性，输入颜色数组：
```json
["#ff0000", "#ffff00", "#00ff00", "#00ffff", "#0000ff"]
```

### 3. 径向渐变效果

1. 选择组件
2. 在属性面板的"渐变配置"中将"渐变类型"改为"径向渐变"
3. 自动应用径向渐变效果

### 4. 添加文字阴影

1. 选择组件
2. 在属性面板的"渐变配置"中启用"文字阴影"开关
3. 调整阴影颜色、模糊度和偏移量

### 5. 调整文字样式

在属性面板的"渐变配置"中调整：
- **字体大小**：8-200px
- **字体粗细**：normal/bold 或 100-900
- **字间距**：调整字与字之间的距离
- **行高**：0.5-3 倍行高
- **文本对齐**：左对齐/居中/右对齐

## 常见用途

1. **仪表板标题**：创建炫彩的数据可视化标题
2. **数据展示**：突出重要数据指标
3. **装饰元素**：增强页面视觉效果
4. **品牌展示**：展示品牌特色文字效果

## 技术实现

组件使用 CSS 的 `background-clip: text` 和 `WebkitBackgroundClip: text` 实现渐变文字效果，支持所有现代浏览器。

## 注意事项

1. 渐变颜色数组至少需要 2 种颜色
2. 文字内容支持多行文本（使用换行符）
3. 阴影效果可能影响性能，建议在必要时启用
4. 渐变角度仅在线性渐变时有效
